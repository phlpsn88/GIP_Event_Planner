# Event planner analyseverslag

## Doel?
Duidelijke, overzichtelijke en gebruiksvriendelijke weergave van events & planning.

## Voor wie?
Organisatoren en mensen die een gemakkelijkere manier zoeken om events bij te houden.

## Functionele analyse
> homepagina (uitleg, "welkom (username))
- Account aanmaken, inloggen
> login pagina
> registreerpagina
- Events
    - Aanmaken
    - Bekijken
    - Aanpassen
    - Verwijderen
    > overzichtspagina (titel, beschrijving, datum, locatie, status weergeven + "edit"-knop + "verwijder"-knop)
    > event aanmaken
    > event aanpassen
